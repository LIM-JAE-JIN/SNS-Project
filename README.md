## 환경변수

`.env` 파일 생성 후 아래 내용 입력 (env-keys.const.ts 파일 참조)

```
PROTOCOL= http
HOST= localhost:3000
DB_HOST= DB 엔드포인트 주소
DB_PORT= DB 포트번호
DB_USERNAME= 사용자 명
DB_PASSWORD= 비밀번호
DB_DATABASE= DB 이름
HASH_ROUNDS= 로그인 비밀번호 해시 생성 강도
JWT_SECRET= JWT 생성 및 검증 키
```

## 실행 방법

```sh
yarn install
yarn start:dev
```
